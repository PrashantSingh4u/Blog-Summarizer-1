# 🧠 Blog Summarizer Chrome Extension

An AI-powered Chrome Extension that extracts blog content from any webpage and generates a concise summary using an AI API.

## 📌 Features

- Generates AI-powered summaries

## ⚙️ Installation (Development Mode)

1. Clone this repository
2. Open Chrome and go to:
   chrome://extensions/
3. Enable "Developer Mode"
4. Click "Load Unpacked"
5. Select the project folder

The extension will now appear in your browser.

---

## 🚀 How It Works

1. User clicks the extension
2. Content script extracts page text
3. Text is sent to AI API
4. Summary is displayed in popup
